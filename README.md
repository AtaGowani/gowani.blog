# gowani.blog | My personal blog for tech, philosophy, and theology

## Environment
* `ruby 3.0 =>`

## Starting out
### Install dependecies locally
```
bundle config set --local path 'vendor/bundle'                        
bundle install
```
## Serve locally
```
bundle exec jekyll serve
```
